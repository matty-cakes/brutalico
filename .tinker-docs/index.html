Accordions

Accordions

You will also notice that Brutalico, like Pico, has accordions, but they are quite different.

Of particular note this is the first material change to Pico I'm making and I'm def nervous about it. The difference from Pico in this case is that I want the user to use BOTH a single <summary> element for the accordion header area and as many <section> elements as they need for elements within thr accordion body.
